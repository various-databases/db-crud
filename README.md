## 项目简介

### 项目起因

本项目主要提供Java操作数据的几种方式示例。

### 项目框架

`db-jdbc`: 基于JDBC实现

`db-dhcp`: 基于数据库连接池实现

`db-mybatis`: 基于Mybatis框架实现

`db-hibernate`: 基于Hibernate框架实现

`db-druid`: 基于阿里巴巴数据库连接池实现

`db-shell`: 基于脚本实现


> **备注:** 示例持续更新中。
